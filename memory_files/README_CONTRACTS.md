## Key Operational Decisions Embedded in This System

1. **$250 Work Order Cap** — This is the line that separates Brenda's approval authority from owner approval. Stated in PMA 1.2, referenced in Vendor 2.4, highlighted in Quick Start.

2. **15% Vendor Markup** — Standard AIRG margin on materials. Stated in Vendor 2.1. All invoices must show this separately. If vendor disagrees, they're not a preferred vendor.

3. **10% Late Fee (Rent)** — CA-standard, CA-defensible. Stated in Lease 3.1. Apply uniformly to all tenants (fair housing).

4. **21-Day Deposit Refund Deadline** — Non-negotiable CA law. Miss this = lawsuit. Stated in Lease 3.8. Alex must hit this religiously.

5. **48-Hour Invoice Due Date** — Keeps cash flow & reconciliation on track. Stated in Vendor 2.3. Brenda enforces.

6. **Cannot Charge Tenant for Notice Service** — CA law violation. Stated in Lease 3.6 (marked CRITICAL). If Gary ever sees this happening, reverse immediately + apologize.

7. **Insurance Certificate Required Before Work** — Risk management. Stated in Vendor 2.2. No cert = no work order issuance.

8. **Owner Liable for Habitability Issues** — If owner refuses repair >5 days, AIRG escalates to attorney. Stated in PMA 1.3. Protects AIRG legally.

## How to Keep This System Updated

### Monthly
- **No action required.** This system is reference material, not a living log.

### Quarterly (with Gary)
- Review for outdated language or changed terms
- Add any new clauses Gary has negotiated with owners/vendors
- Flag any CA law changes (rent control, tenant protection, eviction rules change frequently)
- Update "Last Updated" date at top of each file

### When a Legal Dispute Arises
- Post-dispute, add to SECTION 6 "Common Violations" so future staff avoid the same issue
- Example: If a vendor sues for payment after we held funds for overbilling, document lesson learned

### Annually (Year-End)
- Full audit: Confirm PMA template reflects current terms with all active owners
- Confirm vendor agreement reflects current 15% markup policy + insurance minimums
- Confirm lease template reflects current CA law (especially rent control, just cause, notice requirements, deposit caps)

## What This System Does NOT Replace

1. **Actual executed contracts** — This is a template/reference guide. Always check the actual PMA/lease signed with the specific owner/tenant if there's a dispute.

2. **Legal counsel** — This summarizes CA law as it existed in Q4 2025, but CA law changes constantly. For novel situations, escalate to Gary + Joshua Clark (attorney).

3. **Business judgment** — This system tells you the rules. You still have to apply judgment (e.g., "Should we sue this tenant for $500?" is business judgment, not a clause).

4. **Negotiation authority** — This system reflects AIRG's current standard terms. If an owner asks for different fee structure or a vendor wants different payment terms, that's Gary's call to negotiate.
